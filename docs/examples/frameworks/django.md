## Packages
we are going to install the following packages
* `pyright` `html-lsp` `css-lsp` `json-lsp`  - LSP
* `black` `djlint` `prettier`                - Formatter
* `python` `html` `css` `json`               - Treesitter (Parser)


# Installation
